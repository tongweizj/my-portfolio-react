## 3. 组件重构：容器组件 vs 展示组件

这是 React 最经典的重构逻辑：

- **展示组件 (UI)**：

  只负责样式，数据通过 `props` 传入。例如 `ArticleCard.jsx`。

- **容器组件 (Logic)**：

  负责调用 API 钩子、处理业务逻辑。例如 `ArticleListContainer.jsx`。

- **收益**：

  当你以后想把“草原新闻网”的风格改掉时，你只需要动 UI 组件，逻辑层完全不动。

. 拆分展示与逻辑 (features/articles/ & pages/)
动作：将原来的大页面拆开。

例子：pages/ArticlePage.jsx 只负责获取 URL 里的 ID，而真正的渲染交给 features/articles/components/ArticleDetail.jsx。
